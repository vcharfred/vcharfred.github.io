资讯
发布日期:   2021-02-28

17999元起，华为Mate X2折叠屏手机来了！鸿蒙2.0马上就到

牛年第一部「牛机」来了！

刚刚，华为发布了新一代折叠旗舰 Mate X2。这是春节过后，第一场重磅的旗舰手机发布会。
为什么说它是「牛机」呢？截至发稿前，Mate X2 在华为官网的预约人数已经超过了 240 万人，难抢程度可以预料。

说回华为 Mate X2 本身，作为华为第三代折叠屏手机，和前两代机型 Mate X、MateXs 相比，它最大的变化便是折叠方案的改进：由外折变成内折。

和外折相比，内折方案明显的优势在于屏幕可以被更好地保护，不易刮花或磨损。但此前的内折方案普遍存在一个问题，就是手机折缝大，且易出现折痕。

不过华为表示，Mate X2 采用了新的技术与材料。折缝与折痕，将不再是内折方案的痛点。

在抖音爆火被播放了4亿多次的换脸App，是能有多魔性？

大家还记得在 GitHub 上曾经红极一时的项目：Avatarify 吗？这玩意来自一位俄罗斯的老毛子程序员 Ali Aliev 之手，受疫情的影响，Ali Aliev 被憋在家中整天开视频会议，由于实在是太过无聊了，他就打起了 AI 换脸的主意。

他开发了一个名为 Avatarify 软件，能够帮你在视频会议中帮你换脸，伪装成大佬的样子，比如币圈大佬马斯克。
这个项目被他发布 GitHub 上之后，立刻受到了国外网友们的热捧，才上线十天，就登上了 GitHub 的日榜第一。

不过对于国内的用户来说，Avatarify 这个换脸软件还是比较陌生的，毕竟想要换脸，你必须在电脑上进行一定程度的机器学习，门槛是相对比较高的。

好在，最近这款软件已经被移植到了 iOS 端并且进行了一些优化，几乎是零门槛的操作，让这款魔性的软件瞬间登顶了 iOS 摄影录像榜的第一。注意由于国内网络原因，很多特效无法下载，需要运用科学上网工具帮哈忙哦。

并且还因为这款 App 里一个魔性的动效：蚂蚁呀嘿。被国内的各种短视频软件给玩坏了，仅仅抖音上这个#蚂蚁呀嘿的超话就有 4.4 亿的播放。

点进去之后，那是一片群魔乱舞的“ 祥和 ”景象，甭管你在外面混的有多牛，在这个超话下都得乖乖的唱一句：蚂蚁呀嘿。

游戏官网上线，字节跳动能成中国游戏第三极吗？

字节跳动和腾讯的竞争愈发贴身肉搏。

2 月 22 日，字节跳动又放出一个大瓜，旗下游戏官网——朝夕光年正式上线。在同一天，字节跳动因不服福州中院将“字节诉腾讯不正当竞争案”移送深圳法院审理的管辖权裁定，表示在本月初向福建高院提起上诉，目前上诉已获受理。

据彭博社估计，字节跳动 2020 年全年收入约 1750 亿元，其中游戏板块业务收入为 40 亿-50 亿，如以 40 亿计算，这个成绩将超越畅游，排进中国十大游戏公司之列。

“狗咬吕洞宾，不识好人心” 的典故

其实大家都误解可爱的狗狗了，“狗咬吕洞宾，不识好人心” 中的“狗咬”叫苟姚，话说当年苟姚还是一个贫困的读书人，吕洞宾和苟姚很好就一直在帮助他并且让他住在自己家里；
后来苟姚结婚的时候，吕洞宾要求苟姚让他的新婚妻子陪伴自己几天，苟姚很气愤但是迫于无奈还是同意了，后面苟姚的妻子回去后说相公真是太努力了；苟姚问了缘由才知道原来
吕洞宾那几天其实什么都没有做，还是告诉他的妻子说他在勤奋的读书。苟姚听后才知道自己误会吕洞宾了，至此以后他更加努力的读书最后高中当上了大官。
多年以后吕洞宾家发生变故，吕洞宾去找苟姚帮忙，但是苟姚却让他回去；同时有派人送了一副棺材到吕洞宾家，吕洞宾到家后发现家里挂满了白灯笼，妻子看到他以为他复活了，
吕洞宾打开送回来的棺材看到里面全是金银财宝，由此才明白是自己误会苟姚了。后来人们将这个事情传送，最后就变成了“狗咬吕洞宾，不识好人心”
