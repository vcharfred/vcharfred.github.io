MySQL安装和配置

数据库 MySQL
数据库
发布日期:   2020-08-22

在Windows系统下安装MySQL，使用的是压缩版解压安装，虽然稍微麻烦点，但是不会安装其他的东西。

一、下载MySQL安装包

访问MySQL官网：https://www.mysql.com/；这里我们安装使用压缩包来安装，原因是当你想卸载它的时候你就会感受到了；同时由于我们基本都是用于日常开发测试，因此建议使用压缩包来安装。

下载步骤如下：

  • 前往下载页：

  • 选择社区版：

  • 版本选择：这里选择Windows版本的压缩包下载

  • 进入开始下载页，这里不用去注册，直接下载即可

    二、配置和安装MySQL

    配置

  • 解压刚刚下载的压缩包，将其放在你想安装的位置；尽量不要有中文路径。

  • 在安装目录下（如：我的在C:\dev\db\mysql8016）创建一个my.ini的文件；在里面写入如下配置：

[mysql]
# 设置mysql默认字符集
default-character-set=utf8 
[mysqld]
#设置3306端口
port = 3306 
# 设置mysql的安装目录
basedir=C:/dev/db/mysql8016
# 设置mysql数据库的数据的存放目录
datadir=C:/dev/db/mysql8016/data
# 允许最大连接数
max_connections=200
# 服务端使用的字符集默认为8比特编码的latin1字符集
character-set-server=utf8
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB
[client]
# 设置mysql客户端默认字符集
default-character-set=utf8
  • 配置系统环境变量

a.选择高级环境变量–>>环境变量

b.在 path 变量中添加 mysql的路径(mysql根路径\bin)

安装和启动

  • 安装之前先初始化MySQL；找到cmd命令符程序右击 以管理员身份运行 在命令行中进入mysql目录下的bin目录下
    执行mysql服务安装命令:
mysqld install

成功后会提示：Service successfully installed.

注意：如果不是以管理员身份运行的话出现这个错误：Install/Remove of the Service Denied!

  • 初始化数据库
#这个命令会生成一个随机密码
mysqld  --initialize
# 由于我们需要这个密码，因此使用这个命令让其显示
mysqld --initialize --console


如果没看到，可以将初始化的data目录删除，重新运行命令

也可用户这个 mysqld –initialize-insecure，生成一个以root为用户名，空密码的用户数据

  • 启动MySQL服务
net start mysql

  • 登录数据库
mysql -uroot -p

之后输入密码即可登录

  • 修改密码
use mysql；
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '新密码，必须包含大小写字母、数字、特殊字符';
FLUSH PRIVILEGES;
  • 关闭MySQL服务
net stop mysql

安装启动中遇到的问题

  • 使用启动命令时提示MySQL 服务无法启动

这个可能是没有执行初始化命令。

三、卸载

使用msi方式安装的直接像卸载软件一样直接卸载就可以了 ：）。

使用安装包的卸载方法如下：

mysqld --remove mysql
